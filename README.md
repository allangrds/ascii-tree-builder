# ASCII Tree Builder

## Descrição

Ferramenta web para criar e visualizar estruturas de pastas e arquivos em formato ASCII. Permite adicionar, editar e organizar pastas e arquivos de forma visual, gerando automaticamente a representação em árvore ASCII com os caracteres de box-drawing (├─, └─, │).

## Como instalar e executar

\`\`\`bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev
\`\`\`

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
